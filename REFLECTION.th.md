# บันทึกสะท้อนการทำงาน

ภาษาไทย | [English](REFLECTION.md)

ส่วนที่ยากที่สุดสำหรับผมคือการทำให้ Prisma 7 และ SQLite ใช้ได้บน Windows
โดย Prisma schema engine จะจัดการ schema ได้ก็ต่อเมื่อมีไฟล์ฐานข้อมูลอยู่ก่อน ทำให้ตอนเริ่มจาก clean checkout แบบปกติไม่ได้
ผมจึงเพิ่ม wrapper สำหรับตรวจสอบและสร้างไฟล์ SQLite ตาม path ที่กำหนดโดยไม่ลบฐานข้อมูลเดิม แล้วเรียก Prisma CLI ด้วย URL แบบ absolute
จากนั้นแยกฐานข้อมูลทดสอบและอนุญาตให้ reset ได้เฉพาะ `backend/test.db` เพื่อป้องกันคำสั่งทดสอบจะไปลบข้อมูลสำหรับพัฒนา
งานส่วนนี้ทำให้ผมเข้าใจว่า setup script ต้องตรวจ path ทำซ้ำได้ และมี tests เหมือนกับ API endpoint
